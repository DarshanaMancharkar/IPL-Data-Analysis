# IPL-Data-Analysis
Interactive IPL Data Analytics Dashboard built using Python (Pandas, NumPy) and Power BI with DAX, analyzing team, batting, bowling, and match performance (2008–2024).

## 📌 Project Overview

This project uses IPL datasets to perform data cleaning, exploratory data analysis (EDA), and interactive dashboard development.

The dashboard provides insights into:
            - 📊 Overall IPL statistics
            - 🏆 Team performance
            - 🏏 Batting analysis
            - 🎯 Bowling analysis
            - 📈 Season-wise trends
            - 🪙 Toss decision analysis

## 📈 Key Insights

- Mumbai Indians are among the most successful IPL teams.
- Virat Kohli is one of the highest run scorers while Chris Gayle has hit the most sixes.
- Most teams prefer to field after winning the toss.
- IPL scoring trends have increased over the years.
